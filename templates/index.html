API Endpoints

1. Get a Country-Specific Random Address

GET /api/address?code=

GET /api/address?name=

Use code for country codes (e.g., US, BD) or name for country names (e.g., United States, Bangladesh).

Returns a random fake address for the specified country. If the identifier is misspelled, it will provide a suggestion.

2. Get Available Countries and Data Counts

GET /api/countries

Returns a JSON object with available country codes, their full names, and the number of datasets for each.

3. Get a Fully Random Address

GET /api/random

Returns a random fake address from any available country.
